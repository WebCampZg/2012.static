Bruno Kovačić: Što sve pruža Windows Azure?

Tweet

Opis teme

Windows Azure je već dobro poznata Microsoftova cloud computing platforma, prisutna na tržištu već preko dvije godine. S obzirom da je cloud computing danas vruća tema, iz dana u dan možemo vidjeti razne novitete kod svih cloud computing providera, a tako i kod Microsofta. Windows Azure je tijekom ove godine dobio gomilu novih funkcionalnosti, čime je znatno povećao svoju konkurentnost na tržištu.

Azure danas nudi različite PaaS i IaaS modele, od koji neki ciljaju na velike cloud korisnike, a neki direktno konkuriraju klasičnim hosterima. Uz sve to, Azure je potpuno otvorena platforma. Osim .Neta, podržane su tehnologije poput PHP-a, Jave, node.js-a i Pythona.

Na predavanju će biti prikazani različiti scenariji korištenja Windows Azurea, sa različitim tehnologijama i operativnim sustavima, te usporedba njegovih mogućnosti s konkurentskim cloud i on-premise uslugama.

O predavaču

Bruno Kovačić je osnivač tvrtke Axilis, tvrtke specijalizirane za razvoj programskih rješenja temeljenih na Microsoft platformi. Kroz posljednjih 5 godina razvijao je desktop, web i mobile aplikacije, a u zadnje vrijeme najviše se orijentirao na razvoj Windows Azure aplikacija. Programiranjem se bavi još od osnovne škole, kada je počeo aktivno sudjelovati na raznim informatičkim natjecanjima.
