Saša Jurić: Erlang u praksi

Tweet

Opis teme

Erlang, open source platforma nastala u Ericssonu, u zadnje vrijeme se sve češće koristi za izradu skalabilnih, pouzdanih, visoko opterećenih serverskih sustava. Ovo predavanje služi kao uvod u ovu, pomalo egzotičnu, platformu, prikazuje njene glavne tehničke karakteristike i posebnosti, te dodatno uključuje osvrt na osobno iskustvo pri razvoju http push servera koji u produkciji poslužuje do 2000 requesta u sekundi.

O predavaču

Saša Jurić, developer u minus5, prve linije koda napisao je davne 1991. godine, a deset godina kasnije otisnuo se u profesionalne vode. Prvi dio karijere proveo je razvijajući Windows desktop aplikacije koristeći C++/MFC te C#/.NET. Nakon toga se prebacuje na izradu Ruby on Rails + Javascript baziranih web i kiosk aplikacija. U zadnje dvije godine postaje veliki štovatelj i intenzivni korisnik Erlang platforme.
